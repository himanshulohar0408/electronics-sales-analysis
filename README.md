# Electronics Retail Sales & Operations Analysis (2024–2025)

## Overview
This project analyzes electronics retail sales data to identify revenue drivers, profit trends, customer behavior, and sales patterns using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
- Time period: 2024–2025
- Domain: Electronics Retail
- Data includes product, pricing, profit, customer, and payment information.

## Key Analysis
- Category-wise and brand-wise revenue analysis
- Discount vs profit evaluation
- Customer behavior analysis (New vs Returning)
- Payment mode preference analysis and charts

## Key Insights
- Laptops and Home Appliances are major revenue contributors
- Moderate discounts improve sales without heavily impacting profit
- Returning customers contribute significantly to total revenue
- Card payments are the most preferred payment mode

## How to Run
1. Clone the repository
2. Open `analysis.ipynb`
3. Run all cells
